How It Works

6-Step Inference Pipeline

1
Parse BUFR

Decode binary radiosonde data into a pressure-level DataFrame using eccodes.

2
Clean Profile

Remove sentinel values (−9999, 2e20) and sort levels surface → top.

3
Burst Detection

Apply WMO 30 hPa ceiling threshold to classify as premature or nominal.

4
Feature Engineering

Compute shear, Richardson number, icing index, moisture depth and 20+ features from the vertical profile.

5
Load Saved Model

Read the trained classifier, scaler, label encoder and feature column list from disk.

6
Predict & Explain

Scale features, run ML inference, and generate evidence-backed explanation with full probability breakdown.
